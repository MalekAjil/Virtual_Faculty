اسم الشهادة التخصص المتطلبات مجالات المعرفة الوظيفة الامتحانات الرسوم
MCP (Microsoft Certified Profissional) اي تخصص
 حسب التخصص حسب التخصص حسب التخصص اي إمتحان من إمتحانات ميكروسوفت 125 دولار
MCSA (Microsoft Certified System Administrator) شبكات خبرة في إدارة و صيانة انظمة شبكات تعمل على Windows Server
 مدير شبكات أربعة,ثلاثة إجبارية وأخر اختياري 125 دولار للإمتحانات
MCSE (Microsoft Certified System Engineer 2000 Track)
 شبكات تحليل متطلبات العمل , تخطيط و تشغيل أنظمة الشبكات محلل شبكات, مهندس شبكات, مستشار تقني سبعة امتحانات 125 دولار للإمتحانات
MCSD (Microsoft Certified Solution Developer)
 برمجة و تحليل خبرة لا تقل عن سنتين تحليل و تصميم حلول للأعمال المتوسطة الحجم و الكبيرة. محلل نظم, مهندس برمجيات, مصمم ومخطط نظم خمسة إمتحانات 125 دولار للإمتحان
MCAD (Microsoft Certified Application Developer)
 برمجة خبرة لاتقل عن سنة برمجة و صيانة برامج وتطبيقات, ويب أو ويندوز مبرمج, مطور برمجيات ثلاثة إمتحانات 125 دولار للإمتحان
MCDBA (Microsoft Cerified Database Administrator)
 قواعد بيانات
 سنة خبرة تصميم و إدارة بيئة عمل MS SQL Server مدير قواعد بيانات 4 إمتحانات
 125 دولار للإمتحان
MCT (Microsoft Certified Trainer)
 غير محدد خبرة سنتين في مجال التدريس غير محدد مدرس, مدرب راجع الرابط
 راجع الرابط
